### JavaScript 사용법

자바스크립트는 웹페이지를 동적이고 상호작용 가능하게 만드는 데 사용되는 핵심 프로그래밍 언어이다

```js
const myButton = document.getElementById("myButton"); // 버튼 요소 가져오기
const messageParagraph = document.getElementById("message"); // 메시지 표시할 p 태그 가져오기-------------------

// 2. 이벤트 리스너 추가 (버튼 클릭 시)
myButton.addEventListener("click", function () {
  // 버튼이 클릭되면 실행될 코드
  console.log("버튼이 클릭되었습니다!"); // 콘솔에 로그 출력
  messageParagraph.textContent = "버튼을 눌렀어요!"; // p 태그의 내용을 변경
  alert("Hello, JavaScript!"); // 경고창 띄우기
});

// 3. 변수와 상수 사용 예시
let userName = "개발자"; // 변수 선언 (값 변경 가능)
const PI = 3.14159; // 상수 선언 (값 변경 불가능)

console.log(`안녕하세요, ${userName}님!`); // 템플릿 리터럴을 이용한 문자열 출력
```

### JAVA 사용법

```java
public class VariableExample {
    public static void main(String[] args) {
        String name = "Kimsu"; // 문자열 변수
        int hour = 15;          // 정수 변수
        double score = 90.5;    // 실수 변수
        char grade = 'A';       // 문자 변수
        boolean pass = true;    // 논리(참/거짓) 변수

        System.out.println("이름: " + name);
        System.out.println("시간: " + hour);
        System.out.println("점수: " + score);
        System.out.println("등급: " + grade);
        System.out.println("합격 여부: " + pass);
    }
}
```
